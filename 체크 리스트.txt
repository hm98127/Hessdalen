서버 테스트

1. 첫 페이지에 상품 이미지가 나오는지 확인 해본다.

2. 로그인 버튼을 누르면 로그인 화면이 나오는지 확인 해본다.

3. 회원 가입 버튼을 누르면 회원 가입 화면이 나오는지 확인 해본다.

4. 회원 가입 화면에서 각각 이름, 아이디, 아이디 체크, 비밀번호, 비밀번호 확인, 쓰지 않으면 에러 메시지가 나오는지 확인 해본다.

5. 회원 가입을 한 후 데이터 베이스에 회원 데이터가 올려져 있는지 확인 해본다.

6. 회원 가입한 아이디로 로그인이 되는지 확인 해본다.

7. 로그인 후 로그아웃 버튼을 눌러 로그아웃이 되는지 확인 해본다.

8. 로그인 화면에서 아이디 찾기 버튼을 눌러 아이디 찾기 화면이 나오는지 확인 해본다.

9. 아이디 찾기 화면에서 이름을 적은 후 가입한 아이디를 보여주는지 확인 해본다.

10. 로그인 화면에서 비밀번호 찾기 버튼을 눌러 비밀번호 찾기 화면이 나오는지 확인 해본다.

11. 비밀번호 찾기 화면에서 이름과, 아이디를 적은 후 '비밀번호 변경' 화면이 나오는지 확인 해본다.

12. '비밀번호 변경' 화면에서 비밀번호 적은 후 변경 한 비밀번호로 로그인이 되는지 확인 해본다.

13. 로그인 화면에서 네이버 로그인 버튼과 카카오 로그인 버튼을 눌러 로그인이 되는지 확인 해본다.

14. 네이버 로그인과 카카오 로그인 후 로그아웃 버튼을 눌러 로그아웃이 되는지 확인 해본다.

15. 로그인을 한 후 회원 페이지 버튼을 눌러 '회원 페이지' 화면이 나오는지 확인한다.

16. 회원 페이지 화면에서 가입한 아이디와 이름이 표시 되는지 확인 해본다.

17. 회원 페이지 화면에서 비밀번호 변경 버튼을 누른 후 '비밀번호 변경' 화면이 나오는지 확인 해본다.

18. 비밀번호 변경 화면에서 비밀번호를 적은 후 메인 페이지로 이동되면서 로그아웃이 되어 있는가?

19. 변경한 비밀번호로 로그인이 되는가?

20. 회원 페이지 화면에서 회원 탈퇴 버튼 누른 후 회원 탈퇴 화면이 나오는가?

21. 회원 탈퇴 화면에서 비밀번호를 적은 후 회원 탈퇴 메시지가 나오는가?

22. 회원 탈퇴 후 데이터 베이스에서 실재로 회원 데이터가 지워졌는가?
